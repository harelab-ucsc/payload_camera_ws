# as7265x\_at

ROS2 module for communicating with the AS7265X\_AT multispectral camera by SparkFun.

## Installation

The `as7265x_at_node` ROS2 node relies on the inclusion of the ["as7265x\_at\_msgs" module][https://github.com/harelab-ucsc/as7265x_at_msgs] during module assembly via `colcon`.

## Running the node

```
ros2 run as7265x_at as7265x_at_node
```
